El Desafío

El Problema que Resolvimos

La Situación

En los baños del colegio, algunos estudiantes se tardaban más tiempo del necesario, y los profesores no tenían forma de saber si los estudiantes estaban seguros o si habían regresado a clase.

Nuestra Solución

Creamos un sistema especial de control de acceso usando tarjetas RFID que registra cuándo un estudiante entra al baño y cuánto tiempo permanece fuera del aula. Así ayudamos al colegio a estar más seguro y organizado.

Objetivos del Proyecto

Control de Acceso

Registrar entrada y salida de estudiantes a los baños

Tecnología RFID

¿Qué es RFID?

RFID significa ="Identificación por Radiofrecuencia". Es como una barita mágica que lee información sin tocar nada.

¿Qué es?

Un sistema que usa ondas de radio para identificar objetos o personas de forma automática, sin necesidad de contacto físico.

¿Cómo funciona?

La tarjeta RFID tiene un chip que envía información cuando está cerca de un lector. ¡Es como magia, pero es ciencia!

Ejemplos de uso

Tarjetas de transporte público, pases de acceso a edificios, identificación de mascotas, y ahora ¡en nuestro colegio!

Componentes del Sistema RFID

1

Tarjeta RFID

Cada estudiante tiene una tarjeta con un chip único que lo identifica.

2

Lector RFID

Dispositivo que lee la información de la tarjeta cuando está cerca.

3

Computadora

Procesa la información y registra quién entró, a qué hora y cuánto tiempo.

4

Pantalla LCD

Muestra información en tiempo real del estudiante que acaba de pasar la tarjeta.

Componentes

Materiales Utilizados

Estos son todos los componentes electrónicos y materiales que usamos para construir nuestro sistema

Raspberry Pi 4

Computadora principal

Pequeña computadora que funciona como el cerebro del sistema. Ejecuta el programa y procesa toda la información.

Procesador: Quad-core ARM
Memoria: 2-4 GB RAM

Sensor RFID RC522

Lector de tarjetas

Lee las tarjetas RFID de los estudiantes y envía la información al Raspberry Pi para procesarla.

Frecuencia: 13.56 MHz
Distancia: 0-5 cm

Pantalla LCD 16x2

Display de información

Muestra en tiempo real el nombre del estudiante y mensajes del sistema cuando se pasa la tarjeta.

Tamaño: 16 columnas x 2 filas
Retroiluminación: Azul

Protoboard

Base de conexiones

Tablero donde conectamos todos los componentes sin necesidad de soldar. Permite hacer pruebas y cambios fácilmente.

Cables Jumper

Cables de conexión

Cables para conectar todos los componentes entre sí.

Tarjetas RFID

Identificación de estudiantes

Tarjetas que cada estudiante lleva consigo. Contienen un chip único que identifica a cada persona.

Proceso

¿Cómo Funciona el Sistema?

El sistema funciona en 5 pasos simples que se ejecutan automáticamente

1

El estudiante acerca su tarjeta

Cuando un estudiante necesita ir al baño, acerca su tarjeta RFID al lector que está instalado en la entrada del baño.

2

El lector identifica la tarjeta

El sensor RFID lee el código único de la tarjeta y lo envía al Raspberry Pi para que lo procese.

3

El sistema busca al estudiante

El programa busca en la base de datos qué estudiante tiene ese código de tarjeta y obtiene su nombre y curso.

4

Se registra la hora

El sistema guarda la fecha y hora exacta en que el estudiante entró al baño. Esta información queda almacenada en la base de datos.

5

Se muestra en la pantalla

La pantalla LCD muestra el nombre del estudiante y un mensaje de confirmación. Los profesores pueden ver en tiempo real quién está fuera del aula.

Fundamentos científicos y tecnológicos del proyecto

RFID (Radio Frequency Identification)

Tecnología de identificación mediante ondas de radio que permite la comunicación inalámbrica entre un lector y una etiqueta o tarjeta. Opera en diferentes frecuencias, siendo 13.56 MHz (High Frequency) la más común para aplicaciones de control de acceso.

Computación de Propósito General en Hardware de Bajo Costo

El Raspberry Pi representa una revolución en la educación tecnológica al ofrecer capacidades de computación completas en un dispositivo accesible. Permite ejecutar sistemas operativos completos y controlar hardware a través de sus pines GPIO (General Purpose Input/Output).

Sistemas de Bases de Datos Relacionales

Uso de MySQL para almacenar y gestionar información de estudiantes y registros de acceso. Las bases de datos relacionales permiten consultas eficientes y mantienen la integridad de los datos mediante relaciones entre tablas.
